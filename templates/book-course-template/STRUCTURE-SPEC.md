# Book Course Structure Specification (ENFORCEMENT)

## Purpose
Defines the REQUIRED internal structure for every Old Testament book course.

This is a governance-enforced schema, not a suggestion.

---

## Required Files (MANDATORY)

Every `OTBK-XXX-BookName-Explained/` folder MUST include:

- README.md
- syllabus.md
- module-map.md
- textbook-alignment.md
- playlist-links.md

---

## Optional but Recommended

- assignments.md
- assessments.md
- grading-rubric.md
- learning-outcomes.md
- seat-time.md
- accreditation-notes.md

---

## Folder Layout

OTBK-XXX-BookName-Explained/
├── README.md
├── syllabus.md
├── module-map.md
├── textbook-alignment.md
├── playlist-links.md
├── assignments.md
├── assessments.md
├── grading-rubric.md
├── learning-outcomes.md
├── seat-time.md
└── accreditation-notes.md

---

## Compliance Rules

- Missing required files = NON-COMPLIANT
- Partial templates = NON-COMPLIANT
- Placeholder text must be clearly labeled

---

## Enforcement

- Checked during audits
- Required for accreditation submission
- Required for course activation in course-instances/

---

## Future Automation

AI agents will:
- scan folders
- validate structure
- flag missing files
