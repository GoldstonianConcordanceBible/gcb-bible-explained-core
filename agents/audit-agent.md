# Audit Agent

## Purpose
Simulates external accreditation audit.

---

## Audit Checklist

1. Governance present
2. Routing defined
3. Course structure compliant
4. Metadata complete
5. Accreditation framework present
6. Student tracking exists
7. Evidence available

---

## Output Format

Audit Report:

- Overall Status: PASS / FAIL
- Risk Level: LOW / MEDIUM / HIGH

---

## Example

FAIL:
- Missing syllabus in 12 courses
- Metadata mismatch in 3 entries
