# Goldstonian Concordance Bible (GCB) — Bible Explained Core
## SydTek University | SydTek Scholars | GCB Theology Spine

This repository contains the foundational theology course architecture for the Goldstonian Concordance Bible (GCB) instructional ecosystem.

It serves as the entry point into the Bible curriculum at SydTek University by organizing three core courses:

1. BIBL 101 – The Bible Explained: Structure, Story, and Divine Narrative
2. BIBL 201 – The Old Testament Explained: Law, Covenant, and Prophecy
3. BIBL 202 – The New Testament Explained: Christ, Church, and Eternal Kingdom

These courses are designed to convert YouTube playlists into academically structured course experiences supported by companion textbooks, assignments, assessments, learning outcomes, and accreditation-ready documentation.

## Institutional Purpose

This repository supports:

- course delivery through playlist-based modular learning
- textbook alignment for each foundational course

## Repository Structure

- `/program-overview/` — vision, philosophy, and curriculum overview
- `/courses/` — full course shells for the three foundational courses
- `/textbooks/` — textbook concepts, front matter, outlines, and metadata
- `/playlists/` — playlist registry and mapping to modules
- `/accreditation/` — standards alignment, seat-time tracking, and assessment framework
- `/metadata/` — machine-readable repository and course metadata
- `/assets/` — graphics, diagrams, branding, and supporting visuals
