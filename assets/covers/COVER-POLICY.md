# Cover Image Policy

## Purpose
Ensure every course has a valid visual asset without creating false completion signals.

---

## Rule 1 — Every course must have a cover

If final cover not available:
→ Use placeholder

## Rule 3 — Placeholder Protocol

- generic GCB cover
