# Cover Generation Pipeline

## Style Requirements
- Landscape orientation
- Ultra hyperrealistic
- Renaissance painting style
- Blackest black + goldest gold
- Biblical texture

---

## Prompt Template

"Ultra hyperrealistic Renaissance-style painting of [Book Theme],
dramatic lighting, black and gold color palette,
ancient biblical atmosphere, cinematic composition,
landscape orientation, 16:9"

---

## Workflow

1. Generate placeholder for all courses
2. Replace with AI-generated covers
3. Update cover-registry.json
4. Commit changes
