# Student Record

This folder contains the complete course participation record for one student in one specific course instance.

## Included Files
- `profile.json`
- `enrollment.json`
- `progress.json`
- `time-log.json`
- `quiz-results.json`
- `assignment-status.json`
- `completion.json`

## Accreditation Use
These records support:
- attendance/progress documentation
- learning verification
- assessment evidence
- completion and credit awarding
