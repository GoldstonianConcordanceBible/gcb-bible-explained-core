# Course Sequencing Map

## Foundational Spine

### Stage 1 — Entry Layer
- BIBL 101 – The Bible Explained: Structure, Story, and Divine Narrative

### Stage 2 — Canon Division Layer
- BIBL 201 – The Old Testament Explained: Law, Covenant, and Prophecy
- BIBL 202 – The New Testament Explained: Christ, Church, and Eternal Kingdom

### Stage 3 — Book-by-Book Expansion
- OT book courses
- NT book courses
- Ethiopian canon expansion courses

### Stage 4 — Advanced Thematic and Doctrinal Courses
- wisdom theology
- prophecy theology
- covenant theology
- Christology
- ecclesiology
- eschatology
- Mirror → Water → Fire interpretive seminars

## Program Logic

BIBL 101 establishes the whole-story framework.  
BIBL 201 and BIBL 202 divide that framework into two canonical movements.  
Book-by-book courses deepen content mastery.  
Advanced seminars synthesize doctrine, interpretation, and application.
